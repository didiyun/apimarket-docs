## 接口描述
请求路径：`https://mapapi-places.apigw-gz.didiyunapi.com/api/v1/place/routesearch`

请求方法：GET
## 输入参数

| 参数名称      | 必填 | 默认值 | 说明                                                         |
| -------------- | -------- | ------ | ------------------------------------------------------------ |
| route      | 是       | 无     | 路途经纬度串：lng,lat&#124;lng,lat&#124;…。经纬度小数点后不超过6位。至少输入3个点（包含起终点，至少3个） |
| categories | 是       | 无     | 搜索POI类型编码或名称，最多5个，类型之间“&#124;”分隔，详见滴滴POI类型列表。也支持品牌和风味编码。输入时POI分类编码请添加大写字母 C 前缀；品牌分类编码请添加大写字母 B 前缀；风味分类编码请添加大写字母 F 前缀。 |
| location   | 否       | 无     | 当前车位置：lng,lat。                                        |
| city       | 否       | 无     | 只返回city指定城市的结果。支持省、市中文、全拼、adcode格式输入。暂不支持全拼输入 |
| page_index | 否       | 1      | 当前页数，最多100页，最多返回1000个结果。                    |
| page_size  | 否       | 20     | 每页POI个数，最大50，最多返回1000个结果                      |
| extensions | 否       | false  | 除返回POI基本信息外，是否返回详细信息，返回详细信息收费更高。 |
| children   | 否       | 0      | 指定显示子POI的方式：0（归类到父POI之中，折叠显示），1（子POI单独显示）；目前只支持0 |

## 输出参数
|参数名称  | 类型 | 描述|
|--------|-----|-----|
|status | int  |状态码 |
|info|string|状态说明	|
|result | [Result](#Result)|请求返回数据 |
